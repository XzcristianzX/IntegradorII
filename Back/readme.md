## API Endpoints Propietario

<code>GET /user </code> - Todos los usuarios

<code>POST /user </code>-  Registro de usuario

<code>PUT /user /otro</code> - Logeo de usuario

<code>PUT /user/:id </code> - ruta para actualizar usuario
